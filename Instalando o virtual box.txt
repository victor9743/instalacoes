Instalando o virtual box

Primeiro, remova a instalação parcial do VirtualBox:
sudo dpkg --purge virtualbox-7.1

Instale as dependências necessárias:
sudo apt-get update
sudo apt-get install -f
sudo apt-get install libqt6core6t64 libqt6dbus6t64 libqt6gui6t64 libqt6help6 libqt6printsupport6t64 libqt6statemachine6 libqt6widgets6t64 libqt6xml6t64 libtpms0

Agora instale o VirtualBox corretamente:
sudo apt-get install ./Downloads/virtualbox-7.1_7.1.12-169651~Ubuntu~noble_amd64.deb (baixar o arquivo antes)


- se der erro acima, rodar estes comandos:
